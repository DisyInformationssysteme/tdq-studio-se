Exporter des modèles :

Cet assistant exporte plusieurs expressions régulières dans un fichier csv.

Le format du fichier est le suivant :
  • C1: Label: libellé du modèle (ne doit pas être vide)
  • C2: Purpose: objectif du modèle (peut être vide)
  • C3: Description: description du modèle (peut être vide)
  • C4: Author: auteur de l'expression régulière (peut être vide)
  • C5: Relative_Path: chemin d'accès relatif au dossier racine (peut être vide)
  • C6: All_DB_Regexp: expression régulière applicable à toutes les bases de données (peut être vide)
  • C7: DB2_Regexp: expression régulière applicable aux bases de données DB2 (peut être vide)
  • C8: MySQL_Regexp: expression régulière applicable aux bases de données MySQL (peut être vide)
  • C9: Oracle_Regexp: expression régulière applicable aux bases de données Oracle (peut être vide)
  • C10: PostgreSQL_Regexp: expression régulière applicable aux bases de données PostgreSQL (peut être vide)
  • C11: SQL_Server_Regexp: expression régulière applicable aux bases de données SQL Server (peut être vide)
  • C12: Sybase_Regexp: expression régulière applicable aux bases de données Sybase (peut être vide)
  • C13: Ingres_Regexp: expression régulière applicable aux bases de données Ingres (peut être vide)
  • C14: Informix_Regexp: expression régulière applicable aux bases de données Informix (peut être vide)
  • C15: SQLite3_Regexp: expression régulière applicable aux bases de données SQLite3 (peut être vide)
  • C16: Teradata_Regexp: expression régulière applicable aux bases de données Teradata (peut être vide)

Note :
Le fichier CSV exporté peut être utilisé pour importer les modèles dans un autre Talend Open Studio for Data Quality.
